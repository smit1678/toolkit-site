Form design

Creating ODK forms

Forms can be created using spreadsheet software (such as Excel or LibreCalc) or using the ODK Form Builder. Documentation on how to design a form can be found here.

  • Example ODK form
  • Blank ODK form

Creating OMK forms
Forms for OMK are slightly altered from ODK forms to incorporate the OSM tagging scheme and need to be built using spreadsheet software (such as Excel or LibreCalc). In addition to the structure of an ODK form, OMK forms require an additional ‘osm’ tab that alters the form structure for OMK. Documentation on how to design a form can be found here.

  • Example OMK form
  • Blank OMK form

Form conversion

After forms are developed, they need to be converted from .xlsx/.xls to .xml to be used by the ODK application. This can be done by using XLSform online or offline.
